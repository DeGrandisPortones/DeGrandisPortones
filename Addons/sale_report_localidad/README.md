# Ventas - Localidad y Provincia en Analisis

## Que hace
Este modulo agrega y expone campos en `sale.report` para que puedas:

- agrupar el analisis de ventas por localidad
- filtrar por localidad
- ver la localidad en la vista de lista del reporte
- agrupar el analisis de ventas por provincia
- filtrar por provincia
- ver la provincia en la vista de lista del reporte

### Origen de los datos
- **Localidad**: `res.partner.city`
- **Provincia**: `res.partner.state_id`

## Instalacion
1. Copia la carpeta `sale_report_localidad` dentro de tu ruta de addons custom.
2. Actualiza la lista de aplicaciones.
3. Instala o actualiza el modulo **Ventas - Localidad y Provincia en Analisis**.

## Uso
Ir a:
**Ventas -> Reportes -> Ventas**

Luego en el buscador vas a tener:
- el campo **Localidad** para filtrar
- el campo **Provincia** para filtrar
- la opcion **Agrupar por -> Localidad**
- la opcion **Agrupar por -> Provincia**
