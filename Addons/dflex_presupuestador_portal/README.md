# Dflex Presupuestador Portal (Odoo 18)

## Objetivo
Presupuestador accesible desde Portal/Website para distribuidores (sin licencias internas), con cálculo:
- Precio Distr. = precio según pricelist del Website (idealmente: 'Predeterminado')
- Precio final = Precio Distr. * (1 + coeficiente/100)

## Instalación
1) Copiar `dflex_presupuestador_portal` al addons path
2) Actualizar lista de Apps
3) Instalar módulo
4) Crear rubros en: Presupuestador > Rubros
5) En Portal: /my -> Presupuestador
